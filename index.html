// How this works
This tool uses the WebCodecs API + Canvas to process video entirely in your browser — no upload needed. For large files or maximum compatibility, the tool generates a ready-to-run FFmpeg command you can run locally.

FFmpeg equivalent:
ffmpeg -i input.mpg -vf "crop=ih*5/4:ih,scale=1280:1024" -c:a copy output.mp4
